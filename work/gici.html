GICI

UX Design | Html | Css | Javascript | Wordpress

GICI needed a new website to raise awareness for Gastro-Instestinal Cancer, and Cancer research in New Zealand. The design needed to be fresh and informative, having a welcoming vibe was also important to insure users weren't scared off by the seriousness of the content.

This site needed to be user friendly for all ages, young and old ,so a universal structure was used to keep the site easy for all clientele. I was responsible for the design and build of this site. The site runs off Wordpress so that it is easily updated by the client.
